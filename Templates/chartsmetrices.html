Charts & Metrices



Univariate Graphs

Univariate graphs are used to visualize data distributions and characteristics of a single variable. They help in understanding the central tendency, dispersion, and shape of the data. Below are some common types of univariate graphs we are providing:


Histogram

Histogram represents the frequency distribution of a continuous variable. It divides the data into intervals (bins) and shows the number of observations falling into each interval.

Box plot

A visual summary of data distribution, displaying the median, quartiles, and outliers, providing insights into central tendency and variability.

Violin plot

A blend of a box plot and a kernel density plot, showing the distribution of the data along with its probability density, offering a comprehensive view of data distribution.

Dot-plot

A simple yet effective method to visualize individual data points along a single axis, facilitating easy identification of patterns, outliers, and trends.

insert_chart_outlined

Quantitle-quantile plot

A graphical tool to compare the distribution of two datasets, assessing if they originate from the same population distribution or not.

multiline_chart

Density plot

A smoothed representation of the distribution of a continuous variable, aiding in understanding the shape and pattern of data distribution.

scatter_plot

Destribution plot

A comprehensive visualization illustrating the distribution of data, often through kernel density estimation or histogram representation, enabling easy interpretation of data spread and central tendency.




Bivariate Graphs

Bivariate graphs are used to visualize the relationship between two variables. They help in understanding the correlation, patterns, and trends between the variables. Below are some common types of bivariate graphs we are providing:


bubble_chart

Scatter plot

Visualizes relationships between two variables by plotting points on a Cartesian plane, making it easy to identify correlations or patterns in data.

Line plot

Illustrates trends over time or sequential data points, with data points connected by lines, providing a clear depiction of changes and patterns.

Bar chart

Displays categorical data with rectangular bars of lengths proportional to the values they represent, facilitating easy comparison between different categories.

Pie Chart

Represents parts of a whole by dividing a circle into sectors, each corresponding to a proportion of the total, aiding in visualizing percentages or proportions.

Correlation matrix

Depicts the correlation coefficients between multiple variables in a tabular format, allowing for quick assessment of relationships and dependencies between variables




Multivarite Graphs

Multivariate graphs are used to visualize relationships between multiple variables simultaneously. They help in understanding complex patterns and interactions among variables. Below are some common types of multivariate graphs we are providing:


gradient

Heatmap

Visualizes data in a matrix format using colors to represent values, making it easy to identify patterns and variations across two dimensions

scatter_plot

3D scatter plot

Extends the scatter plot into three dimensions, enabling visualization of relationships among three variables, often used for exploring complex data relationships.

timeline

Parallel coordinates plot

Represents multivariate data by plotting each variable along a separate axis and connecting points corresponding to each data point, facilitating comparison and pattern recognition across multiple variables.

perm_identity

Andrews plot

Represents multivariate data by transforming each data point into a curve based on the coefficients of a Fourier series, enabling visualization of patterns and differences between groups.

face

Chernoff faces

Utilizes facial features to represent multivariate data variables, where different facial characteristics such as eyes, nose, and mouth represent different variables, allowing for intuitive visualization and comparison.

poll

Radar chart

Displays multivariate data on a circular plot with each variable represented by a spoke, making it easy to compare the relative values of multiple variables across different categories or groups.




Other Types of Charts

Here is the descriptions of some other types of charts:


view_module

Treemap

Visualizes hierarchical data structures using nested rectangles, with each rectangle representing a category or item and its size corresponding to a quantitative value, enabling easy comparison of proportions within the hierarchy.

donut_large

Sunburst chart

Represents hierarchical data in a radial layout, where each ring represents a level of the hierarchy, and segments within rings correspond to categories or items, aiding in the visualization of hierarchical structures and proportions.

all_inclusive

Force-directed graph

Illustrates relationships between entities using nodes and links, where nodes represent entities and links represent relationships, with the layout determined by simulated forces between nodes, facilitating exploration and visualization of complex networks.

linear_scale

Sankey diagram

Depicts flows and relationships between entities using directed arrows of varying widths, showing the magnitude of flow between nodes, making it useful for visualizing processes, flows, and relationships.

grid_on

Covariance matrix

Displays the covariance values between multiple variables in a tabular format, providing insights into the relationships and dependencies among variables.
